Första lektionen

Olika termer

HTML= Hyper Text Markup Language. Det är språket man använder för att skapa webbsidor. Den består av text, länkar, bilder och bestämmer hur texten ska se ut.
W3C= World Wide Web Consortium. Det är organisationen som bestämmer över standarer som ska användas. De standarer handlar om att alla webbutvecklare ska följa samma riktlinjer. Det är viktigt för annars är det svårt för webbläsarna att läsa olika sätt att skapa programm eller webbsidor.
webbserver= Det är en server som levererar en webbsida när en webbläsare skickar begäran om ett URL.
client= det är webbläsaren som begär att få se en webbsida.
API= Det är en webbsida där man kan hämta information för att sätta i sin egen webbsida.
URL och vad den består av. Första delen kallas för protokoll-http://, subdomän-www., server-webbsidans namn, domäm-.com, och sökvägen till websidans sida.

HTML

Html består av taggar som också kan kallas element. De olika taggarna gör att texten blir uppdelad i titlar och stycken. Taggarna består oftast av två stycken. En starttagg, och en sluttagg. Sluttaggen inehåller det här tecknet: /. I taggarna kan det finnas attribut som avgör textens storlek-font-size: 12px;, färg-color:red;, emphasis-em och mycket mer.

Man ska alltid starta en html sida med taggen !DOCTYPE html.
Html består också av head och body sektionen. Head sectionen inehåller taggen title som anger sidans titel som står i webbläsaren. Body sektionen inehåller allt som man kan se på webbsidan.

Html, CSS, Javascript

Html är webbsidans inehåll och srtuktur, CSS är ansvaring för webbsidans design och layout och Javascrit gör att man kan tillsätta olika funktioner som calender, tid och mycket mer.

Block element vs Inline element

Block element börjar alltid på en ny rad och sträcker sig till ända slutet av sidan medans inline elementet tar så mycket plats som det behövs.
